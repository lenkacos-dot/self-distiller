# Self-Distiller — 价值体系（Value System）

> 对你来说，什么重要、什么不重要、什么绝对不能妥协？
> 价值体系是数字生命的"道德罗盘"——决定了 AI 在虚拟空间中替你做选择时的底线和优先级。

---

## 维度 03：价值体系

描述用户的价值观、道德观、优先级排序。不一定是"宏大的道德哲学"，而是日常选择中体现出来的权重体系。

---

## 子维度

| 子维度 | 涵盖内容 |
|--------|---------|
| 核心原则 | 你的人生信条、座右铭、反复强调的原则 |
| 优先级排序 | 家庭 vs 事业 vs 自由 vs 安全——你真正把什么排第一 |
| 道德边界 | 什么是你绝对不做的、什么可以妥协 |
| 公平感 | 对分配公平、机会公平的看法；被不公平对待时的反应 |
| 群体归属 | 对集体/团队/家庭的忠诚度 vs 个人主义倾向 |
| 时间和金钱观 | 省钱 vs 省时间、及时行乐 vs 储蓄规划 |
| 审美评判 | 你觉得什么是"好的"什么是"丑的"（不只是外表，也包括行为） |

## 查询示例

- "我的人生优先级是什么？排个序"
- "我在钱的问题上是什么态度？"
- "有什么事情是我绝对不能接受的？"
- "我理想的生活是什么样的？"
- "按我的价值观，我应该怎么处理这个道德困境？"

---

## 数据来源

价值体系可以从以下材料中提取：

1. **聊天记录中的表态** — 对事件、人物的评论中体现的价值观
2. **消费记录** — 花钱的优先级（为知识付费？为体验付费？）
3. **社交媒体签名/简介** — 自己选择的标签和宣言
4. **日记/反思** — 自我对话中的价值判断
5. **简历/求职信** — 自己描述的职业追求和动力来源
